# TRIVIA
### Paso 1:
- clonar el repositorio
`git clone https://github.com/AlbeCamp21/Prueba_entrada_CC3S2.git`

### Paso 2:
- Dirigirse al repositorio
`cd Prueba_entrada_CC3S2`

### Paso 3:
- Dirigirse a trivia-game-python/
`cd trivia-game-python/`

### Paso 4:
- Levantar el contenedor
`sudo docker-compose up --build`

### Paso 5:
- En otra ventana, en la misma ubicación, dirigirse a app/
`cd app/`

### Paso 6:
- Ejecutar la trivia
`python3 trivia.py`
